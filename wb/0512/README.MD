### 中间件
1. 大坝 => 拦截水 => 沉砂池 => 净化水 => 导流隧洞 => 发电厂
2. express 的最大特色就是中间件，也是最重要的一个设计，就是中间件。、
3. 一个 express的应用就是由许多的中间件来完成的。
4. express 中间件 和 AOP 面向切面是一个意思。就是需要经过一些步骤，不去修改自己的代码，以此来扩展一些功能。
5. AOP面向切面 可以想象为一个工厂的流水线。
    - 5.1 采摘 => 清洗 => 包装，如果说增加了工序 采摘 => 分类 =》 清洗 => 贴标签 =》包装，这成了一个新的流程，工人是可以随时撤回的。
    - 5.2 AOP面向切面，比如：日志记录，性能统计，安全控制，事物处理等。从业务逻辑中要划分出来，通过对这些行为的分离，我们希望要把这些行为独立出来，不影响业务代码的逻辑。
    - 5.3 我通过加减功能，不影响原有的功能。


### next
1. 中间函数中调用了next函数之后，就会调用下一个中间件函数
2. next 就会匹配下一个匹配的规则 => 请求方式 + 请求地址
3. 当前没有执行完中间件函数，就不会匹配下一个规则，就会挂起 peeding。
4. 中间件是由步骤的

* 中间件的编写位置会影响到下一个匹配的规则

### 携带参数和判断
1. 携带参数使用 request.xxxx = xxxx, 那么匹配到的中间件就可以通过 request.xxxx 来获取属性值。
2. 中间件可以进行响应数据，通过 response 来继续响应。
